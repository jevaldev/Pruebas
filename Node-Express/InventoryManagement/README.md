# Inventory Management

A simple backend for inventory management made with ExpressJS and MySQL. This project is for learning/practice backend and Express knowledge. I'm also try to apply the MVC (Model-View-Controller) architecture to keep the project a little bit more organized.

## Technologies

---

- **Node.js**
- **Express.js**
- **MySQL**
- **zod**

## Features

---

This project would let you:

- [ ] Create products.
- [ ] List products.
- [ ] Filters and paginate products results.
- [ ] Update products information.
- [ ] Change the status of a product (active/inactive).
- [ ] Delete products.
- [ ] Register inventory movements (stock in/out)
- [ ] Mantain a log of the inventory movements
